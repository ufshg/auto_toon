# Auto Toon!

입력으로 주어진 사진을 빛의 가산효과를 이용해 만화 스타일로 변경해주는 코드입니다.

### 사용방법

1. openCV, Numpy 설치
2. 폴더에 사용할 이미지 이름을 _image.jpg_ 로 지정해 저장
3. `autotoon.py` 실행
4. 폴더에 생성된 _output.jpg_ 를 확인
